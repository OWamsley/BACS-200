How to Get Started

Yoyoing is a very easy hobby to get into. All you need is a yoyo, and an internet connection.


Which Yoyo should you buy?

When starting out yoyoing, you will want a yoyo that is responsive, and not un-responsive. Responsive yoyos will come back to your hand with a tug, while unresponsive yoyos require a trick called a "bind" to bring back to hand. Unresponsive yoyos tend to spin longer and make string tricks easier, but beginners will be able to start yoyoing way easier with a responsive yoyo.

There are many good choices for a first yoyo, but my recommendation is to go with a YoyoFacory Replay. You can find a responsive version easily, and you can eventually switch out it's bearing to make it unresponsive as you get better. You can find it here on the yoyoexpert website.


Where should I learn tricks?

There are multiple places that you can learn tricks. It will probably be easier to stick to one place, as you can learn tricks in an ordered manner, where the next should be a little bit harder than the last trick. I recommend using YoyoExpert, or using yoyotricks.com


All that's left is to go out there and learn some tricks, and practice!
